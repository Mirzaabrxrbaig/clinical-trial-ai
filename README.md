# Clinical Trial Patient Matching System

AI-powered healthcare platform that matches patients with relevant clinical trials using NLP, semantic search, and vector databases.

## Features
- AI patient-trial matching
- Semantic healthcare search
- Responsive healthcare UI
- React frontend
- FastAPI backend
- Vector similarity search
- Dynamic routing
- Mobile responsive design

## Technologies Used
- Python
- FastAPI
- React.js
- FAISS
- Sentence Transformers
- NLP
- REST API
